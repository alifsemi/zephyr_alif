.. _alif_e4_dk:

Alif E4 Development Kit
#######################

.. warning::
   This board is deprecated and will be removed in a future release.
   E4 series SoCs are now supported on the Alif E8 Development Kit (alif_e8_dk) board.
   Refer to the E8 Development Kit documentation for board identifiers and
   build instructions.

Overview
********

The Alif E4 is a cost-optimized variant of the Ensemble E8 SoC with Generative AI capabilities.

.. important::
   The :ref:`alif_e8_dk` is the primary development platform for E4, E6, and E8
   SoC variants. Please refer to the E8 Development Kit documentation for detailed
   hardware specifications, programming instructions, and supported features.

   The E4 and E6 are scaled-down versions of the E8 with reduced features.

Board Identifiers
*****************

For building applications targeting the E4 SoC:

- High-Efficiency core: ``alif_e4_dk/ae402fa0e5597xx0/rtss_he``
- High-Performance core: ``alif_e4_dk/ae402fa0e5597xx0/rtss_hp``
