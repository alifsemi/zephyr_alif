Alif E3 Development Kit
#######################

Overview
********

The Alif E3 is a cost-optimized variant of the Ensemble E7 SoC.

.. important::
   The :ref:`alif_e7_dk` is the primary development platform for E3, E5, and E7
   SoC variants. Please refer to the E7 Development Kit documentation for detailed
   hardware specifications, programming instructions, and supported features.

   The E3 and E5 are scaled-down versions of the E7 with reduced features.

Board Identifiers
*****************

For building applications targeting the E3 SoC:

- High-Efficiency core: ``alif_e3_dk/ae302f80f55d5xx/rtss_he``
- High-Performance core: ``alif_e3_dk/ae302f80f55d5xx/rtss_hp``
